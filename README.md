# Sekhon Car Rentals

Premium car rental website built using Flask, HTML and CSS.

## Features

- Delhi round trip services
- One way travel
- Himachal tours
- Wedding Fortuner booking
- Self drive SUVs
- WhatsApp direct booking
- Responsive modern UI

## Technologies Used

- Python Flask
- HTML5
- CSS3

## Run Locally

```bash
pip install -r requirements.txt
python app.py
